Waterwall

by Mattia Barbiero

A Cakephp component to detect web attacks like Xss, Sql injection, FLI and others.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/barraemme/waterwall



Install

  • 1 Have a working CakePHP 1.2 App (maybe work on 1.3 ?).
  • 2 Download Waterwall archive and unzip the contents on app folder.
  • 3 Include the component in any controller that will need it (and set params if necessary).
    Simple include: 
        	var $components = array('Waterwall');
    Custom include: 
        	var $components = array(
    	       'Waterwall' =>
    	 	 array(
    		   'xss' => array(
		     'level' => 1, //1 => basic,2 => medium,3 => paranoic - 0 to disable
		     'source' => 'REQUEST', // POST or GET or REQUEST,
		     'callback' => false 
		   ),
		   'sql' => array(
		     'level' => 1, //1 => basic,2 => medium,3 => paranoic - 0 to disable
		     'source' => 'REQUEST', // POST or GET or REQUEST 
		     'callback' => false 
		   ),
		   'file' => array(
		     'level' => 1, //1 => basic,2 => medium,3 => paranoic - 0 to disable
		     'source' => 'FILES', // FILE
		     'callback' => false 
		   ),
		   'autoscan' => true
		)
	);
